Kickstarter Tableau Dashboard

Tableau Visualization Project | Crowdfunding Data Analysis

Project Overview

This Tableau project analyzes Kickstarter crowdfunding campaigns using an interactive dashboard. The dashboard presents project outcomes, pledged funding trends, geographic backer distribution, and pledge amount segmentation.

The goal of the project is to demonstrate how Tableau can transform raw crowdfunding data into clear visual insights that help understand campaign performance and funding behavior.

Dashboard Components

  • Distribution of projects by campaign status: successful, failed, canceled, live and suspended.
  • Global map showing backers and average pledge amount by country.
  • Pledge amount segmentation by funding ranges.
  • Quarterly pledged amount trend with moving average analysis.

Tools Used

Tableau • Excel • Data Visualization • Interactive Dashboard Design • KPI Analysis

Key Insights

  • Failed projects represent the largest share of campaigns.
  • Successful projects make up a significant portion of the dataset, showing clear differences between funded and unfunded campaigns.
  • Pledged amounts changed over time, with clear quarterly trends and moving-average patterns.
  • Backer activity and average pledge amounts vary across countries.
  • Pledge ranges help identify how funding is distributed across different campaign sizes.
